Modul 1: Konsep Dasar Algoritma

🧠 Apa itu Algoritma?

Algoritma adalah urutan langkah-langkah logis untuk menyelesaikan suatu masalah.

📝 Karakteristik Algoritma:

  • • Input: Data yang diproses
  • • Output: Hasil yang diharapkan
  • • Definiteness: Langkah yang jelas
  • • Finiteness: Berakhir dalam waktu terbatas
  • • Effectiveness: Dapat dilaksanakan

🔄 Contoh Algoritma dalam Python

Algoritma Mencari Nilai Terbesar:

# Algoritma mencari nilai terbesar
def cari_nilai_terbesar(angka_list):
  terbesar = angka_list[0]
  for angka in angka_list:
    if angka > terbesar:
      terbesar = angka
  return terbesar
# Contoh penggunaan
nilai = [85, 92, 78, 96, 88]
print(cari_nilai_terbesar(nilai))

def bandingkan_angka():

🔍 Algoritma Pencarian

Linear Search

def linear_search(arr, target):
  for i in range(len(arr)):
    if arr[i] == target:
      return i
  return -1

Kompleksitas: O(n)

Keunggulan: Sederhana, bekerja pada array tidak terurut

Binary Search

def binary_search(arr, target):
  left, right = 0, len(arr) - 1
  while left <= right:
    mid = (left + right) // 2
    if arr[mid] == target:
      return mid
    elif arr[mid] < target:
      left = mid + 1
    else:
      right = mid - 1
  return -1

Kompleksitas: O(log n)

Keunggulan: Sangat cepat, efisien untuk data besar

📊 Perbandingan Performa

Untuk array dengan 1000 elemen:

  • • Linear Search: Maksimal 1000 perbandingan
  • • Binary Search: Maksimal 10 perbandingan
